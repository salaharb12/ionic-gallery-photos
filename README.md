# Ionic Photo Gallery

A simple photo gallery app built with Ionic Framework and Angular. This application allows users to capture photos using the device camera and save them to the device's storage.

## Features

- Capture photos using the device camera
- Save photos to the device's storage
- View saved photos in a gallery
- Delete photos from the gallery
